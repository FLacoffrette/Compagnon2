Detailed Description

Application visant à permettre au Linkit One l'envoi d'un SMS contenant la date ainsi que le message qui lui auront été envoyées par une requête HTTP.

Author
Franck L.
Version
1.0
Date
22 fevrier 2018

LIBRAIRIES:

  • LWiFi -> Permet la gestion du WiFi pour le linkit One
  • LWiFiUdp -> Librairie MediaTek permettant la manipulation de paquet UDP afin de traiter des requêtes/réponses au serveur indiqué
  • ArduinoJson -> Contient le parser JSON qui sera utilisé afin de récoltées les données transmisses par le raspberry pi

Function Documentation

◆ loop()

loop ( )

Bouclage infini qui permet au client de se connecter.

  • Permet aux client de se connecter sur le serveur http
  • Disocie l'entête http afin de ne conserver que les données extraites du fichier json les données doivent se trouver dans l'ordre suivant :
  • CONTACT DATE MESSAGE

La taille du buffer et l'allocation ont été allouées à l'aide de l'assistant correspondant à la librairie JSON https://arduinojson.org/assistant/ Une fois les données extraites la connexion est fermée afin de rendre le serveur disponible pour une éventuelle nouvelle connexion

ALGORITHME : Bouclage tant qu'un client connecté a des données à envoyer :

  • Lecture et affichage caractère par caractère des données transmises par JSON
  • Disociation de l'entête HTTP afin de ne conserver que les données JSON
  • Bouclage lors de la lecture/stockage des données du JSON dans le tableau json[]
  • - Parsing des données
  • Si des données ont été récupérées
  • - Récupération de la date et de l'heure
  • - Envoi du SMS contenant la date et le message
  • Fermeture de la connexion
